# Splunk Lab Básico

Repositorio de aprendizaje práctico sobre Splunk Enterprise.
Este proyecto documenta paso a paso la instalación, uso básico y creación
de dashboards en Splunk, desde la perspectiva de un estudiante.

Objetivo:
- Aprender Splunk desde cero
- Documentar el proceso de aprendizaje
- Crear una guía práctica y sencilla para principiantes
